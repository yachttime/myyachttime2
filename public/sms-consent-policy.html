SMS Notification Consent Policy

Proof of Consent Documentation for Twilio A2P Registration

Last Updated: February 9, 2026 | Document Version: 1.0

Overview

This document serves as proof of consent for SMS messaging under Twilio A2P (Application-to-Person) requirements and TCPA (Telephone Consumer Protection Act) compliance. It describes how Utopia Yacht Management collects, stores, and honors SMS consent from users.

Compliance Standards:

TCPA Compliant Twilio A2P Express Written Consent Opt-Out Support

SMS Use Case

Purpose

Work shift reminders for employees

Message Type

Transactional notifications

Frequency

Up to 2 messages per workday

Recipient Type

Employees and staff members

Consent Collection Method

Consent is collected through our web application during user profile creation and management. Users must actively opt-in by checking a consent checkbox before SMS notifications are enabled.

Consent Collection Location

  • Platform: Web application (user management dashboard)
  • Accessible by: System administrators, managers, and the users themselves
  • Consent Type: Express written consent via checkbox
  • Pre-checked: No (unchecked by default)

Consent Tracking & Data Storage

All consent information is stored in our Supabase database with the following fields:

  • sms_consent_given: Boolean flag indicating consent status
  • sms_consent_date: Timestamp of when consent was granted
  • sms_consent_method: Method of consent collection (e.g., "web_form")
  • sms_consent_ip_address: IP address at time of consent (optional)
  • notification_phone: Phone number for SMS delivery

Data Retention: Consent records are maintained for the duration of the user's employment plus 3 years for compliance audit purposes.

Required Disclosures

Our consent form includes all required TCPA disclosures:

  1. Purpose: "receive SMS text message reminders for my scheduled work shifts"
  2. Frequency: "Message frequency varies (up to 2 per workday)"
  3. Cost Disclosure: "Message and data rates may apply"
  4. Opt-Out Instructions: "Reply STOP to opt out at any time"
  5. Voluntary Consent: "Consent is not required as a condition of employment"

Opt-Out & Revocation

Users can revoke SMS consent at any time through multiple methods:

  • Text "STOP": Reply with STOP to any SMS message
  • Dashboard: Uncheck the SMS consent checkbox in their user profile
  • Contact Support: Request SMS opt-out via email or phone

When opt-out is received, the system immediately updates sms_consent_given to false and stops all SMS messaging to that user.
